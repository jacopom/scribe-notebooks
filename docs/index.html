What it does

  • Native desktop window for your Kindle Notebook — no browser tabs, no distractions
  • Export to PDF or PNG — all pages, current page, or a custom range
  • Page-aware navigation — reads Amazon's page count and navigates automatically during export
  • Persistent login — sign in once through Amazon's own page, stay logged in
  • Automatic filename — exports use the notebook title by default
  • Cross-platform — macOS (Apple Silicon), Windows, and Linux (build from source)
  • Free and open-source under the MIT license

Setup

"App is damaged" or "could not be verified" on macOS

The app is not notarized by Apple, so Gatekeeper may block it on first launch. After moving the app to your Applications folder, run this once in Terminal:

# Run once after installing xattr -cr /Applications/Scribe\ Notebook.app

Or go to System Settings → Privacy & Security and click Open Anyway.

Sign in to Amazon

Open the app and use the Sign in… option from the Scribe Notebook menu. Login goes through Amazon's own page — your credentials are never seen by this app. If you use a passkey, use a recovery code or temporary password instead.

Linux — build from source

No pre-built binary for Linux yet. Clone via HTTPS (SSH requires configured GitHub keys):

git clone https://github.com/jacopom/scribe-notebooks.git cd scribe-notebooks npm install npm start

FAQ

What is Scribe Notebook?

A free, open-source Electron app that wraps Amazon's Kindle Notebook web interface in a native desktop window on macOS and Windows, with PDF and PNG export added on top.

Is it free?

Yes, completely free. No account, no subscription, no tracking. MIT license. You only need an existing Amazon Kindle account.

Does it access my Amazon password?

No. Login uses Amazon's own sign-in page in an isolated popup window. Scribe Notebook never sees your credentials — only the session cookie is stored locally.

How does the PDF export work?

Click the print icon in the toolbar. Choose PDF or PNG, then all pages, current page, or a custom range. The app automatically reads the notebook's page count, navigates each page, and saves a single file using the notebook title as the filename.

No. This is an independent, unofficial project. Amazon, Kindle, and Kindle Scribe are trademarks of Amazon.com, Inc.

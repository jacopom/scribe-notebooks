## Development

### Prerequisites
- Node.js 18+ 
- npm 9+
- Git

### Setup
1. Clone the repository
```bash
git clone git@github.com:jacopom/scribe-notebooks.git
cd scribe-notebooks
```

2. Install dependencies
```bash
npm install
```

3. Development Mode
```bash
npm run dev
```

### Building
Create production builds for different platforms:

#### Mac
```bash
npm run build:mac
```

#### Windows
```bash
npm run build:win
```

#### Linux
```bash
npm run build:linux
```
